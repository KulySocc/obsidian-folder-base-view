# Changelog

All notable changes to Folder Base View are documented here. The format is based
on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project
adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

The `### Added / Changed / Fixed` headings below render in-app as
**✨ New / 💎 Improvements / 🐞 Fixes** (see [ADR-0003](docs/adr/0003-changelog-single-source.md)).

## [0.2.0] - 2026-06-06

### Added
- **Release notes:** a what's-new dialog now appears once after the plugin updates, summarising what changed since the version you last saw. Re-open it anytime with the *Show release notes* command.

### Changed
- **Settings:** added a *Show release notes on update* toggle so you can opt out of the what's-new dialog.

## [0.1.0] - 2026-06-05

### Added
- Initial release. Click a folder in the file explorer to open its contents as a Bases-powered table of contents — no companion note required.
- Default Index Base, exact-match per-folder overrides, a configurable open trigger, and a one-click recommended base.
